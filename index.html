Projet de classification automatique de dépêches en Java

(2023)

Ce projet consistais en binômes, de créer un programme Java permettant de trier de courtes dépècges selon certaines catégories. Pour celà, nous avons du créer des listes de mots correspondant à chaque catégorie et assigner un score (poid) à chaque mot dans la catégorie en fonction de son appartenance et à sa précision sur le sujet. Par exemple, dans la catégorie science, le mot 'atome' aurais eu un poid fort contrairement au mot 'je'. Nous devions le faire premièrement à la main, puis nous devions automatiser le processus de tri des dépèches et d'ajustement des valeures pour avoir un taux de réussite le plus haut possible

Création d'une base de données PostgreSQL

(2022)

Nous devions en binômes, modéliser et créer une base de données PostgreSQL sur le thème de naufrahe du Titanic. Nous devions pour modéliser la base de données, utiliser des outils comme l'UML et le format Schema Entity Association pour la représenter. Malhereusement, cette étape n'a pas pu directement être implémantée dans la base de donnée du aux nombreuses solutions différentes. Finalement, nous devions interroger les données fournies pour répondre à des questions dur les naufragés du Titanic.
