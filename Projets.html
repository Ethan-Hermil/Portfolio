Projet de scrabble duplicate web monopage

(2023 - Maintenant)

Ce projet est motivé par ma mère qui aimerais pouvoir jouer au scrabble duplicates avec ses élèves lors de remplacement quand le contexte s'y prète. Hébergant mon projet sur GitHub pages, je n'ai donc pas accès au PHP pour passer les informations entre des pages ou même pour effectuer les calculs du jeu. J'ai donc du créer ce site en une seule page HTML accompagnée de JavaScript pour tout les calculs et les différentes scènes du site. Ce projet est donc un défis pour moi apprenant tout juste le JavaScript. Je me heurte donc à des difficultés sur lesquelles je peut passer plusieurs jours pour les régler du à mon manque de connaissances en JavaScript.
